=====================================
COMECE AQUI - 3 OPCOES
=====================================

OPCAO 1: Teste Rapido (2 minutos)
----------------------------------
Passo 1: Abra PowerShell
Passo 2: Digite:
    cd c:\3D\ProgramViewer3D-main
    python examples_train.py

Resultado: Gera 2 arquivos .pts com o trem


OPCAO 2: Visualize o Trem (3 minutos)
--------------------------------------
Apos OPCAO 1, digite:
    python main.py train_example.pts

Resultado: Abre o visualizador com o trem


OPCAO 3: Trem + Sua Nuvem de Pontos (5 minutos)
------------------------------------------------
Digite:
    python train_simulation.py seu_arquivo.pts

Resultado: Trem passando pela sua nuvem!


=====================================
CONTROLES (Use OPCAO 3)
=====================================

K      = Liga/desliga trem
L      = Liga/desliga nuvem de pontos
G      = Liga/desliga grade
+/-    = Aumenta/diminui velocidade
SPACE  = Pausa/retoma trem
R      = Reseta posicao do trem
ESC    = Sair


=====================================
CODIGO PYTHON MINIMO
=====================================

from utils.train_simulator import TrainSimulator

# Criar trem
trem = TrainSimulator(num_wagons=5)

# Obter pontos para renderizar
pontos, cores = trem.get_points()

# Mover o trem
trem.update(dt=1.0)

# Ver informacoes
print(f"Total de pontos: {len(pontos):,}")
print(f"Posicao Z: {trem.get_position():.1f}")
